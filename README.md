# Neovim
1. Install neovim
2. Create symlink for `init.vim` to `~/.config/nvim/init.vim`
3. Open nvim and type `:PlugInstall` to install plugins
4. Check everything is work using `:checkhealth`
5. Use nerd font for icons.
